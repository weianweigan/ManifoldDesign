## 3-16. 快速旋转
### 3-16-1. 功能描述
主要针对多孔特征进行旋转角度的操作，但无界面，每单击一次逆时针旋转90°；
适用环境：零件环境；
### 3-16-2. 功能界面
无
### 3-16-3. 操作方法
1、在模型视图区域，选择一个要旋转的多孔特征；
2、TouchMDesign >单击“快速旋转”命令，每单击一次，被选择的多孔将逆时针旋转90°；
**Tips：选择要旋转的多孔特征后，可多次单击“快速旋转”命令旋转孔腔，每次旋转90°。**
